# 📱 App Repositories

> **Status do Projeto:** Concluído ✔️

## 💻 Sobre o Projeto

O **App Repositories** é um aplicativo Android desenvolvido como parte do **Santander Mobile BootCamp**  (em parceria com a DIO). O principal objetivo deste projeto foi aplicar conceitos modernos de desenvolvimento mobile, focando em criar uma aplicação robusta, escalável e de fácil manutenção.

Através deste app, foi possível explorar na prática o consumo de APIs REST, gerenciamento de estado e injeção de dependências utilizando as bibliotecas mais recomendadas pelo mercado.

## 🛠 Tecnologias e Arquitetura

Este projeto foi desenvolvido utilizando **100% Kotlin** e implementa as seguintes tecnologias e padrões de projeto:

* 
**Arquitetura:** Padrão **MVVM** (Model-View-ViewModel) para uma separação clara de responsabilidades.


* 
**Injeção de Dependências:** **Koin**.


* 
**Networking:** **Retrofit** para a realização de requisições REST.


* 
**Assincronismo:** **Coroutines** (Corrotinas) para gerenciar threads e chamadas assíncronas de forma eficiente.


* 
**Observabilidade e Ciclo de Vida:** **LiveData** e **Componentes da arquitetura do Android**.



## ✨ Funcionalidades

* Integração com API REST para busca e listagem de dados.


* Gerenciamento reativo do estado da interface do usuário (UI).


* Estrutura modularizada facilitando testes e futuras implementações.



## 🚀 Como Executar o Projeto

### Pré-requisitos

* Android Studio (versão mais recente recomendada).
* Emulador Android ou dispositivo físico configurado para depuração.

### Passos para rodar

1. Faça o clone deste repositório:
```bash
git clone https://github.com/YvesDeSa/app-repositories.git

```


2. Abra o projeto no **Android Studio**.
3. Aguarde o Gradle sincronizar todas as dependências do projeto.
4. Clique no botão **Run** (`Shift + F10`) para iniciar o aplicativo no emulador ou dispositivo físico.
